# DecoRent

Сервис аренды декора для мероприятий (MVP) на **Next.js + TypeScript**.

## Что уже есть

- Главная страница с секциями: описание сервиса, преимущества, «Как это работает», CTA.
- Заглушки страниц: `/catalog`, `/request`.
- Базовая UI‑палитра через CSS‑переменные + Tailwind.

## Как запустить локально

На машине должен быть установлен Node.js (LTS).

```bash
npm i
npm run dev
```
